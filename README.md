# Raspberry Pi Backup

This repository backs up reusable Raspberry Pi files for the `intyu` Pi.

Current maintained folders:

- `opencv/`: OpenCV digit detection scripts and launchers.
- `LCD驱动/`: MPI3501 3.5 inch LCD installation package, install script, LCD test script, and readable recovery notes.
- `显示切换/`: scripts to switch between LCD framebuffer desktop mode and normal HDMI/KMS big-screen mode.

Current Pi address remembered for this setup:

```text
10.81.42.136
```

Do not store SSH passwords in this repository.
